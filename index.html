Hello there! I'm a fullstack engineer currently based in Central Coast, California. I love JavaScript and use React/Redux for my front-end projects and usually Node.js with Express framework for the backend. I have experience in building production apps with Python and PHP - with Flask and Laravel frameworks. And, I dabble in building mobile apps with React Native.

Before building software, I was immersed in the Data Center world, helping build the infrastructure for AWS. There we encountered problems with having an up-to-date inventory of our assets and so, with a colleague, we started to build a web application to help solve the problem. Soon enough, I realized this is what I wanted to do ... coding.

I got my coding chops at App Academy and completed my B.S. in Computer Science at San Jose State University.
